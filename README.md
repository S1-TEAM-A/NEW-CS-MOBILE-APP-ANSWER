# 에스원 Answer — 현장 CS 모바일 앱 (프로토타입)

보안장비 **VOC(고객의 소리)** 현장 대응을 모바일 하나로 연결하는 CS 앱 프로토타입입니다.
`VOC 확인 → AI 진단·조치 가이드 → 조치 결과 입력 → 완료`까지 한 손으로.

## 🔗 데모 (GitHub Pages)
저장소 **Settings → Pages** 활성화 후 아래 주소에서 접속할 수 있습니다.

```
https://s1-team-a.github.io/NEW-CS-MOBILE-APP-ANSWER/
```

## ✨ 주요 기능
- **VOC 배정함 / 상세 조회** — 전체·긴급·진행중·완료 필터
- **AI 진단 체크리스트**(복수 선택) → **조치 가이드** → **조치 결과 자동 작성**
- 숙련자용 **SKIP** + 음성/텍스트 결과 입력
- **오늘의 동선**(가까운 순 · 드래그로 순서 변경) · **지도 길찾기**(내비 연동 느낌)
- 고객 전화 / 문자 템플릿 / 방문 일정 캘린더
- 토스풍 모션 · **AI 분석 인터스티셜** · 결과 타이핑 효과

## 🖥️ 실행 방법
### 1) 그냥 열기
`index.html` 을 크롬/엣지로 열면 바로 동작합니다. (빌드·설치 불필요)

### 2) 로컬 서버로 보기 (선택)
```bash
python -m http.server 8000
# 브라우저에서 http://localhost:8000
```

## 📱 앱처럼 쓰기
모바일 브라우저에서 데모 주소 접속 → **"홈 화면에 추가"** 하면 아이콘이 생기고 앱처럼 실행됩니다.

## 🛠️ 기술
- 단일 파일 정적 웹앱 (HTML / CSS / Vanilla JS) — 외부 의존성·빌드 없음
- 모바일 우선 UI, PWA/네이티브(웹뷰 래핑)로 확장 가능

## 📌 참고
- 화면에 표시되는 VOC·고객사·담당자 정보는 모두 **데모용 가상 데이터**입니다.
- 실제 접수 데이터 및 조치가이드 원본 파일은 **개인정보 보호를 위해 저장소에 포함하지 않았습니다.**
